Logic operators

This is a page with a Tcl script that demonstrates the use of the expr command.

Logic operators

&& Called Logical AND operator. If both the operands are non-zero, then condition becomes true.

|| Called Logical OR Operator. If any of the two operands is non-zero, then condition becomes true.

! Called Logical NOT Operator. Use to reverses the logical state of its operand. If a condition is true then Logical NOT operator will make false.

 

set isAdult 1
set hasLicense 0

if {$isAdult && $hasLicense} {
    puts "You are an adult with a license"
} elseif {$isAdult || $hasLicense} {
    puts "You are either an adult or have a license"
} else {
    puts "You are neither an adult nor have a license"
}
