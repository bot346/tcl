Expressions

This is a page with a Tcl script that demonstrates the use of the expr command.

Arithmetic operators

All the arithemic operations can be performed using command expr

 

set num1 10
set num2 5

set sum [expr $num1 + $num2]
set difference [expr $num1 - $num2]
set product [expr $num1 * $num2]
set division [expr $num1 / $num2]
set remainder [expr $num1 % $num2]

puts "Sum: $sum"
puts "Difference: $difference"
puts "Product: $product"
puts "division: $division"
puts "Remainder: $remainder"
