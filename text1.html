Example 1

Question: A sum of money at simple interest amounts to Rs. 815 in 3 years and to Rs. 854 in 4 years. The sum is:


#given
set SI_3yrs 815
set SI_4yrs 815

# interest per year calculation
set interest_per_year [expr {$SI_4yrs-$SI_3yrs}]

# what is the interest collected for 3 years 
set interest_for_3yrs [expr {$interest_per_year*3}]

# to get principal subtract maturity amount with interest 
set principal [expr {$SI_3yrs-$interest_for_3yrs}]

puts "principal amount is : $principal"
