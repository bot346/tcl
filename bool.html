Boolean

Boolean expressions and values in Tcl are typically used in conditional statements to control the flow of a program. Here are some examples illustrating the use of boolean expressions and values:

Boolean Variables


set num1 10
set num2 20

set isGreaterThan [expr $num1 > $num2]
puts "Is num1 greater than num2? $isGreaterThan"

Basic Boolean Expressions


set x 5
set y 10

if {$x < $y} {
    puts "$x is less than $y"
} else {
    puts "$x is not less than $y"
}

Logical AND (&&) and OR (||)


set isAdult 1
set hasLicense 0

if {$isAdult && $hasLicense} {
    puts "You are an adult with a license."
} elseif {$isAdult || $hasLicense} {
    puts "You are either an adult or have a license."
} else {
    puts "You are neither an adult nor have a license."
}

Negation (!)


set isStudent 1

if {!$isStudent} {
    puts "You are not a student."
} else {
    puts "You are a student."
}

Boolean Variables


set isSunny 1
set isRaining 0

if {$isSunny} {
    puts "It's a sunny day!"
}

if {!$isRaining} {
    puts "It's not raining."
}

Using expr for Complex Expressions


set age 25

if {[expr $age >= 18 && $age <= 30]} {
    puts "You are a young adult."
} else {
    puts "You are not a young adult."
}

Ternary Operator for Boolean Assignments


set score 85
set isPass [expr $score >= 60 ? 1 : 0]

puts "Did you pass the exam? $isPass"
