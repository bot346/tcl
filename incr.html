Increment and Decrement

This is a page with a Tcl script that demonstrates the use of the expr command.

Increment and Decrement

 

set count 5

incr count ;# Increment by 1
puts "After incrementing: $count"
###OUTPUT###
6

set count [expr $count - 1] ;# Decrement by 1
puts "After decrementing: $count"
###OUTPUT###
4
