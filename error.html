Error Handling

This is a simple HTML page with a Tcl script that demonstrates the use of the catch command.

Zero Division Error


#simple division proc
proc division {} {
  set x 11
  set y 0
  set z [expr $x / $y]
}

# catch will give result in 1 or 0 format 
# if result is 1 then there is error
# if result is 0 then there is no error
set result [catch division error_msg]
puts "$result"

if {$result != 0} {
    puts "Error: $error_msg"
} else {
    puts "Result: [division]"
}
