Data Structures

Below codes are for select, slice, show all the content

set x "a b c" 
#select set aa [lindex $x 2] 
puts "Item at index 2 of the list {$x} is: $aa\n" 
#slice 
set y [split 7/4/1776 "/"] 
puts "We celebrate on the [lindex $y 1]'th day of the [lindex $y 0]'th month\n" 
#show all 
set z [list puts "all item in y list $y" ] 
puts "A command resembles: $z\n"
