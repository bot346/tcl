Tcl Expr Example

This is a simple HTML page with a Tcl script that demonstrates the use of the expr command.

    # Tcl code using the expr command
    set num1 10
    set num2 5

    set result [expr {$num1 + $num2}]
    puts "Addition result: $result"

    set result [expr {$num1 * $num2}]
    puts "Multiplication result: $result"
