Variable

set var_name value
If value is specified, then the contents of the variable var_name are set equal to value .
If var_name consists only of alphanumeric characters, and no parentheses, it is a scalar variable.
If varName has the form var_name(index) , it is a member of an associative array.

# x is variable
set X "This is a string"
# y is variable
set Y 1.24
# The dollar sign tells Tcl to use the value of the variable - in this case X or Y.
puts $X
puts $Y
#passing both the variables in single puts 
puts "$X $Y"
