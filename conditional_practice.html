Conditional statements

Basic if statement

This is a simple HTML page with a Tcl script that demonstrates the use of the expr command.

 
    
set age 20

if {$age >= 18} {
    puts "You are an adult."
}

if-else statement

This is a simple HTML page with a Tcl script that demonstrates the use of the expr command.

 
    
set temperature 25

if {$temperature > 30} {
    puts "It's a hot day."
} else {
    puts "It's not very hot today."
}

if-elseif-else statement

This is a simple HTML page with a Tcl script that demonstrates the use of the expr command.

 
    
set score 75

if {$score >= 90} {
    puts "A"
} elseif {$score >= 80} {
    puts "B"
} elseif {$score >= 70} {
    puts "C"
} else {
    puts "Fail"
}

Nested conditions

This is a simple HTML page with a Tcl script that demonstrates the use of the expr command.

 
    
set x 10

if {$x > 0} {
    if {$x % 2 == 0} {
        puts "$x is a positive even number."
    } else {
        puts "$x is a positive odd number."
    }
} else {
    puts "$x is not a positive number."
}

switch conditions

This is a simple HTML page with a Tcl script that demonstrates the use of the expr command.

 
    
set dayOfWeek "Monday"

switch $dayOfWeek {
    "Monday" {
        puts "It's the start of the week."
    }
    "Friday" {
        puts "It's almost the weekend."
    }
    default {
        puts "It's a regular day."
    }
}
