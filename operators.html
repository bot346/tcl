This is a simple page with a Tcl script that demonstrates the use of the relational operators command.

Relational operators 

 

set x 15
set y 20

if {$x == $y} {
    puts "x is equal to y"
} else {
    puts "x is not equal to y"
}

if {$x != $y} {
    puts "x is not equal to y"
} else {
    puts "x is equal to y"
}
